# Bank Application

An example Sber-like banking application.
Application was tested only on **Android**.

## Dependencies

- Flutter: v3.16.4
- Dart: 3.2.3
- openjdk: 18.0.2-ea 2022-07-19
- Android Build Tools: v30.0.2
